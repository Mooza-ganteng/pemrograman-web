Demo: Radio Button, Drop Down, Text Area, & Tata Letak “Frame”

Catatan: tag <frameset> sudah usang (deprecated). Di sini digunakan <iframe> + CSS untuk meniru frame horizontal/vertikal/gabungan.

1) Radio Button

Radio button memungkinkan memilih satu opsi dari beberapa pilihan.

2) Drop Down (Select)

Drop down menampilkan daftar pilihan dalam satu komponen hemat ruang.

3) Text Area

Text area digunakan untuk input teks panjang/multibaris.

4) Frame Horizontal (atas–bawah)

Dua frame ditumpuk vertikal (baris 1 dan baris 2).

5) Frame Vertikal (kiri–kanan)

Dua frame berdampingan dalam dua kolom.

6) Frame Gabungan H–V

Kombinasi: satu frame lebar di atas (horizontal), dua frame di bawah (vertikal).
